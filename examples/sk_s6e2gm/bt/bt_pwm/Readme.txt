Description:

This example uses BT10 as PWM timer to drive the green LED, and in a interval
generated by BT2 (set as reload timer), the duty is modulated. This will
generate the breathing effect on the LED.

================================================================================
How To Use This Code Example 

* Connect board with PC via USB cable
* Run the project.
* Check the status of LED.
